# 🏇 UmaSim - 競馬シミュレーション Webアプリ

実際の金銭を賭けずに、競馬のオッズ読みや資金配分（ベッティング）の戦略をシミュレーションできるWebアプリケーションです。

## 🎯 特徴

- **リアルな競馬シミュレーション**: スピード・スタミナ・コンディションを考慮した本格的なレース展開
- **多彩な券種**: 単勝・複勝・馬連・馬単・3連単に対応
- **AI予想家**: 本命党、穴党、データ派の3人の予想家が買い目を提案
- **詳細な統計**: 的中率、回収率、収支グラフで実力を可視化
- **データ永続化**: LocalStorageで進捗を保存、エクスポート/インポート機能付き
- **レスポンシブデザイン**: PC・スマホどちらでも快適にプレイ可能

## 🚀 技術スタック

- **フレームワーク**: React 19
- **ビルドツール**: Vite 7
- **スタイリング**: Tailwind CSS 4
- **グラフ**: Chart.js + react-chartjs-2
- **ホスティング**: GitHub Pages

## 📦 セットアップ

### ローカル環境での実行

```bash
# 依存関係のインストール
npm install

# 開発サーバーの起動
npm run dev

# ビルド
npm run build

# プレビュー
npm run preview
```

### デプロイ

このプロジェクトはGitHub Actionsを使用して自動的にGitHub Pagesにデプロイされます。

1. リポジトリの Settings → Pages で Source を "GitHub Actions" に設定
2. main ブランチにpushすると自動的にデプロイされます

## 🎮 使い方

1. **ダッシュボード**: 現在の所持金、統計、開催中のレースを確認
2. **レース選択**: 気になるレースをクリックして出走表を表示
3. **AI予想家**: 3人の予想家の見解を参考に
4. **馬券購入**: 券種と馬を選んで購入
5. **レース実行**: リアルタイム実況と共にレースが展開
6. **結果確認**: 的中/不的中と払い戻しを確認

## 📊 ゲームシステム

### 初期設定
- 初期資金: 100万円
- レース: 全4レース（G1, G2, G3）

### 券種
- **単勝**: 1着を当てる
- **複勝**: 3着以内を当てる
- **馬連**: 1着・2着を順不同で当てる
- **馬単**: 1着・2着を順番通りに当てる
- **3連単**: 1着・2着・3着を順番通りに当てる

### レースロジック

各馬の強さスコアは以下の式で計算されます：

```
Score = (Speed × 0.6 + Stamina × 0.4) × Condition × Random(0.9, 1.1)
```

- 短距離（< 1600m）: スピード重視
- 長距離（> 2200m）: スタミナ重視
- 雨天時: スピードの高い馬に不利

## 🎨 デザインコンセプト

競馬場をイメージした緑を基調とした配色に、勝負を象徴するゴールドのアクセントを加えています。
